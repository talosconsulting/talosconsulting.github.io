## PE01-PE04 (4 fully-funded, open positions)

Job description:

This position is for a Platform Engineer for a technical development program supporting the Sponsor’s IdAM/ICAM programs located on a secure network.  As a Platform Engineer on this program, you will design, develop, and implement reliable, robust, and resilient systems used by software development teams.  Demonstratable, working experience with automation tools (e.g., Ansible, Terraform, GitHub Actions), Kubernetes, containers, or serverless is strongly preferred.  The Platform Engineer will work with a large team of developers, software engineers, cybersecurity engineers, database administrators, and system architects. 

Primary responsibilities:
•	This role delivers an innovative product, service, or a combination of products and services that address a customer’s specific requirements
•	Support the development of enterprise software solutions within the customer’s cloud environments
•	This role provides execution guidance and practical engineering support to mission teams

Required qualifications:
* Active TS/SCI with polygraph

Desired qualifications:
•	Demonstratable working experience with automation tools such as Ansible, Terraform, Jenkins, GitHub Actions, and Packer
•	Demonstratable working experience with containers (i.e., Docker/Podman) or their orchestration (e.g., Kubernetes)
•	Demonstratable working experience with serverless platforms (e.g., AWS Lambda)
•	Experience with security accreditation process
•	Experience working in Agile, SAFe, and Scrum environments
